Financial Modeling Snippet


import pandas as pd

df = pd.DataFrame({'Q1': [100], 'Q2': [120]})

# Calculate Quarter-over-Quarter (QoQ) Growth
df['QoQ_Growth'] = (df['Q2'] - df['Q1']) / df['Q1']

print(f"QoQ Growth: {df['QoQ_Growth'].iloc[0]:.2%}")
This code demonstrates our standard process for calculating growth metrics using Python and Pandas. The line numbers are highlighted in sequence to walk stakeholders through the logic. First, we import libraries. Then, we perform the calculation. Finally, we print the result.

Core Financial Formula: ROE

Return on Equity (ROE) measures profitability relative to shareholder equity.

\[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholder Equity}} \]

For Q2:

\[ \text{ROE} = \frac{\$8.1M}{\$55.5M} \approx 14.6\% \]
